# Competitive Intelligence Skill

This skill provides automated competitive intelligence for the Indian smartphone market, with a primary focus on the under ₹30,000 segment. It tracks key competitors like Samsung, Vivo, Motorola, Infinix, Realme, and Redmi, alongside Tecno/Pova products.

## Features

- **Automated Trend Tracking**: Monitors Google Trends across 7-day, 12-month, and 5-year windows.
- **Anomaly Detection**: Identifies significant spikes or drops in search interest for competitor products.
- **Daily Reports**: Generates comprehensive reports summarizing market trends and competitive threats.
- **Configurable Keywords**: Tracks a dynamic list of Tecno products and competitor devices.
- **Actionable Insights**: Provides recommended actions based on competitive movements.

## Scripts

- `scripts/intel_collector.py`: A Python script responsible for gathering competitive data from various sources (e.g., Brave Search API, Google Trends) and processing it. This script is designed to be executable and forms the core logic for intelligence collection.
