# 🚀 Viral Loop Designer Agent

**Purpose:** Turn every customer into a growth engine with viral referral programs, gamification, and word-of-mouth automation.

---

## When to Use

```bash
# Design referral program
@viral-loop-designer create referral

# Build gamification system
@viral-loop-designer gamify

# Create viral campaign
@viral-loop-designer campaign "waitlist-launch"

# Optimize sharing mechanics
@viral-loop-designer optimize

# Track viral metrics
@viral-loop-designer metrics
```

## Capabilities

### 1. Referral Program Design
- Two-sided incentives (referrer + referee)
- Tiered rewards structure
- Viral coefficient optimization
- Attribution tracking
- Fraud prevention

### 2. Gamification Systems
- Points and badges
- Leaderboards
- Achievement unlocks
- Progress bars
- Streak mechanics

### 3. Viral Campaigns
- Waitlist with referrals
- Invite-only launches
- Share-to-unlock content
- Social proof integration
- FOMO mechanics

### 4. Sharing Optimization
- One-click sharing
- Personalized referral links
- Social platform integration
- Message templates
- Preview cards

### Referral Program
```

Program: "Brand Marketing Team Champions"

INCENTIVE STRUCTURE:
Referrer Gets:
• 1 referral = 1 month free
• 3 referrals = 3 months free + swag box
• 5 referrals = Lifetime free access
• 10 referrals = Become affiliate (20% commission)

Friend Gets:
• 20% off first purchase
• Free onboarding call ($200 value)
• Exclusive templates pack

MECHANICS:
✅ Unique referral link per user
✅ Social sharing buttons (email, Twitter, LinkedIn, WhatsApp)
✅ Referral dashboard (track invites, clicks, conversions)
✅ Automated reward delivery
✅ Reminder emails ("You're 1 referral from 3 months free!")

VIRAL BOOSTERS:
• "Most referrals this week" leaderboard
• Double rewards for 48 hours (limited promotions)
• Mystery rewards (gamification)
• Social proof: "John just got 6 months free!"

Expected Viral Coefficient: 0.35
(Each user brings 0.35 new users)
```

