# Copy Editing Skill

This skill is designed to enhance and refine marketing copy, ensuring it is clear, concise, impactful, grammatically correct, and aligned with brand voice. It focuses on improving readability and persuasiveness.

## When to use this skill:
Use this skill when you need to:
- Refine website copy, landing page content, and product descriptions.
- Polish email marketing campaigns, newsletters, and promotional messages.
- Edit social media posts, ad creatives, and blog articles.
- Ensure brand voice consistency across all marketing materials.
- Improve the clarity, grammar, spelling, and punctuation of any text.
- Make copy more engaging and persuasive to drive conversions.

## Capabilities:
- **Grammar & Spelling Correction:** Automatically identifies and corrects grammatical errors, typos, and spelling mistakes.
- **Clarity & Conciseness Enhancement:** Rewrites sentences to improve readability and remove jargon or unnecessary words.
- **Impact & Persuasion Improvement:** Suggests stronger verbs, more engaging adjectives, and persuasive phrasing.
- **Brand Voice Alignment:** Analyzes text against established brand guidelines (if provided) and adjusts tone, style, and vocabulary.
- **Readability Scoring:** Provides metrics on text readability (e.g., Flesch-Kincaid) and suggests improvements.
- **SEO Keyword Integration (basic):** Can suggest minor adjustments for target keywords without compromising natural language flow (advanced keyword integration requires a separate SEO skill).

## Examples:

### Example 1: Edit a piece of landing page copy
```
/copy-editing edit --text="Our product provides great value to customers. It's really good and helps a lot." --tone="persuasive" --brand-voice="energetic"
```
This command would return an improved version of the provided text, making it more persuasive and energetic.

### Example 2: Check an email subject line for impact
```
/copy-editing analyze-impact --text="Important update for you inside!" --type="email_subject_line"
```
This command analyzes the impact and click-worthiness of the email subject line.

### Example 3: Get readability score for a blog post excerpt
```
/copy-editing get-metrics --text="(long blog post excerpt here)" --metric="readability"
```
This command returns the readability score (e.g., Flesch-Kincaid Grade Level) for the provided text.
