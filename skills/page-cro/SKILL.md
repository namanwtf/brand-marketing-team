# 📈 Page CRO (Conversion Rate Optimization)

**Purpose:** Optimize landing pages for maximum conversions using proven frameworks.

---

## CRO Frameworks

### Hero Section
- **Headline:** Clear value proposition in 10 words or less
- **Subheadline:** Specific benefits with numbers
- **CTA:** Action-oriented, contrasting color
- **Social Proof:** Trust badges, ratings, user count

### Above the Fold
- Product image/video
- Key benefits (3 bullet points)
- Price + offer
- Primary CTA

### Trust Signals
- Reviews and ratings
- Security badges
- Return policy
- Warranty information

---

## Optimization Checklist

- [ ] Page loads in < 3 seconds
- [ ] Mobile responsive
- [ ] Single clear CTA
- [ ] No distractions (remove nav)
- [ ] Urgency/scarcity elements
- [ ] Social proof above fold
- [ ] Risk reversal (returns, warranty)
