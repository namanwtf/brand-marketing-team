## ⚡ OpenClaw Installation (30 Seconds)

### Option 1: Direct Skill Installation (Recommended)

```bash
# Install directly into OpenClaw
openclaw skills install https://github.com/namanwtf/brand-marketing-team

# Or clone manually
git clone https://github.com/namanwtf/brand-marketing-team.git ~/skills/brand-marketing-team
```

### Option 2: NPM Global Install

```bash
npm install -g brand-marketing-team
npx brand-marketing-team init my-brand
```

### Option 3: Project Setup

```bash
git clone https://github.com/namanwtf/brand-marketing-team.git
cd brand-marketing-team
npm install
npm run setup
```

**That's it. Your marketing team is awake.**

## 📦 Quick Start Guide

### Step 1: Install

```bash
openclaw skills install https://github.com/namanwtf/brand-marketing-team
```

### Step 2: Configure

```bash
@auto-setup init
# Enter your brand website
# AI discovers: brand voice, competitors, positioning
```

### Step 3: Deploy

```bash
@competitive-intel watch all
@pricing-optimizer auto-adjust --enable
```
